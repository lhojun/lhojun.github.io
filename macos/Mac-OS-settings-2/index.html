    M1 맥 개발환경 설정 - (2) iTerm2 설치 및 설정

    서론

    M1 Pro가 탑재된 2021 맥북프로 16’를 구매해서, 환경설정을 다시 할 필요성이 있었다. 이전 M1 맥북에어 환경설정을 할 땐 이리저리 충돌난걸 ‘야매’로 고쳐서 겨우겨우 돌아가게 한 것들이 많았기 때문에 이번 기회에 깔끔하게 설치도 해보고, 그 기록을 남기고자 한다.

    iTerm2 설치 및 설정

    난 원래 순정 터미널 사용자였지만, 기왕 맥북을 바꾼김에 터미널도 새것처럼 써보자 싶어서 유명한 iTerm2를 깔고 설정을 하게 되었다. 사실 이전에도 몇번 쓰려고 시도해봤는데 생각보다 내가 꾸며놓은게 마음에 들지 않아서 몇번 하차했다가 이번엔 꽤 오래 탑승중이라서 글로 쓰게 되었다. 참고로 이 글은 ‘노마드코더’의 ‘개발자를 위한 M1 풀옵션 맥북프로 세팅!’ 영상을 보고 따라했다! 디테일한 설정만 본인의 취향에 따라 설정하고, 전체적인 진행과정은 같기때문에 영상을 보고 하는 것도 추천한다! (고마워요 니콜라스형!)

    이 글은 아래 과정을 진행했다는 가정하에 진행된다.

    Homebrew가 안깔려있다면 꼭 아래 글을 보고 깔고 오자!

    • M1 맥 개발환경 설정 - (1) 홈브루(Homebrew) 설치
    • iTerm2를 설치하자! (Homebrew 글 마지막에서 설치 과정을 따라한 사람은 생략해도 된다!)
    brew install --cask iterm2

    • 설치가 완료되면 iTerm2를 실행하자.

    아주 밋밋하다.
    • 커맨드(⌘ Command) 키와 쉼표(,)키를 눌러서 환경설정을 열어보자!

    • Appearance 탭으로 이동하자.

    • Theme를 Minimal로 바꿔준다.

    • Profiles탭으로 이동하자.

    • Profiles내의 탭들중에서 Session으로 이동하자.

    • 가장 하단의 Status bar enabled를 체크해준다. (iTerm2에서 상태바를 볼 수 있다.)

    • 체크 후 바로 옆의 Configure Status Bar로 들어가면 보이는 화면이다.

    • 본인이 보고싶은 상태들을 드래그해서 옮기고, 하단의 Auto-Rainbow에서 Color Theme를 설정할 수 있다.

    나같은 경우는 현재 디렉토리 정보, git 상태, CPU 사용량을 담았다. 본인의 취향에 따라 담으면 된다.
    Auto-Rainbow는 Automatic으로 설정해주었다.
    • 이제 iTerm2의 컬러테마를 설정해주자.

    iterm2colorschemes.com으로 이동해준다. iTerm2의 컬러테마를 무수히 공유해주고 있는 사이트이다.
    • 본인이 마음에 드는 컬러테마를 선택하자. 난 Github Dark를 선택했다.

    • 마음에 드는 컬러테마 이름의 링크를 누르고 들어가면 이런 XML들이 뜬다.

    • 해당 파일을 찾기 쉬운 곳에 저장하자. 사파리 기준으로 커맨드(⌘ Command) 키와 알파벳 s(S)키를 누르면 쉽다.

    • 이렇게 txt 파일로 저장이 된다. (굳이 txt파일로 안되었더라도, 아까 사이트에서 본 내용만 깔끔하게 담겨있다면 문제 없다.)

    • 확장자를 txt에서 itermcolors로 바꾸어주자. iTerm2의 컬러테마 파일 확장자이다.

    • .itermcolors 사용을 눌러주자.

    참고로 이 알림이 안뜬다면 (아주 높은 확률로) 확장자를 바꾼게 아니라 파일 이름만 바꾸었을 가능성이 크다.
    그럴 땐 파일을 '우클릭 > 정보 가져오기 > 이름 및 확장자'에서 바로 바꾸어주거나, 확장자 가리기를 체크 해제하고 다시 바꾸면 된다.
    • 잘 바뀌었다!

    • 다시 iTerm2로 돌아와서 Profiles 내의 Colors탭을 선택하고, 방금 받은 컬러테마 파일을 여기로 드래그해주자.

    • 그러면 Colors Scheme이 import되었다고 알려준다.

    • 우측 하단의 Color Presets…를 눌러보면 방금 넣은 ‘GitHub Dark’(혹은 본인이 받은 컬러테마 이름)이 추가되어있다. 클릭해주자.

    • 그럼 색들이 바뀌어서 적용이 된다.

    • 이제 환경설정은 끄고, iTerm2로 돌아가보자.

    아까보다 뭐가 많이 깔끔해졌다. 하지만 끝이 아니다.
    • oh-my-zsh를 깔아주자.

    oh-my-zsh 공식 홈페이지로 이동해준다.
    oh-my-zsh는 zsh의 설정을 조금 더 쉽게 관리하기 위한 프레임워크이다.
    플러그인이나 여러 테마를 적용할 수 있어 더 편리하게 사용 할 수 있다.
    맥 OS 카탈리나 이후부터는 터미널에서 zsh가 기본 설정으로 사용되지만,
    여전히 많은 사람들이 더 편리한 사용을 위해 oh-my-zsh를 설치하고있다.
    누가봐도 눌러야 할 것 같이 생긴 Install oh-my-zsh를 눌러주자.
    • 설치를 할 수 있는 커맨드가 보인다.

    • iTerm2에 붙여 넣어주자.
    sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

    글을 쓰는 현재는 위와 같지만 달라질 수 있으니 홈페이지를 신뢰하자.
    • 억겁의 시간이 흐르고 나면 누가봐도 oh-my-zsh가 설치된 것 같은 화면이 보인다.

    KTX 타고 가면서 봐도 설치된 걸 알 수 있을 수준
    • 그 다음으로 powerlevel10k를 설치해보자.

    powerlevel10k 깃허브로 이동해준다.
    powerlevel10k는 oh-my-zsh에 사용할 수 있는 테마이다.
    나 빼고 다 알고 있는 것 같던데 하마터면 길가다 'powerlevel10k 아세요?' 하는 질문에 '아니요?' 하고 답해서 간첩신고 당할 뻔 했다.
    • 아래로 조금 내리다보면 Installation이 있다.

    누가봐도 눌러야할 것 같은 Manual 밑의 Oh My Zsh를 눌러주자
    • 그러면 Oh My Zsh에서 어떻게 설치해야하는지 알려준다. 첫번째 커맨드를 복사하자.

    • iTerm2에 붙여 넣고 엔터를 쳐주자.
    git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

    글을 쓰는 현재는 위와 같지만 달라질 수 있으니 홈페이지를 신뢰하자.
    • ~/.zshrc에서 테마 설정을 바꾸어주어야 한다. 아래 커맨드를 iTerm2에 입력해서 엔터를 치면 ~/.zshrc가 vscode에서 열린다.
    code ~/.zshrc

    code가 vscode를 여는 커맨드이기때문에, vscode가 안깔려있다면 안 열릴 것이다. vscode를 깔거나, 다른 편집기로 열어주자.
    • ZSH_THEME를 바꾸어주자.
    ZSH_THEME="powerlevel10k/powerlevel10k"

    저장을 잊지말자!
    • ~/.zshrc를 변경하면 터미널을 꼭 재시작해주어야한다.

    터미널을 재실행하면 갑자기 분위기 설문조사가 시작된다.
    첫번째 질문은 powerlevel10k에 아주 적합한 폰트를 설치할것이냐?
    알파벳 y(Y)를 눌러 설치를 하고 진행한다.
    • 폰트를 열심히 설치해주고 있다.

    • 설치가 완료되면, iTerm2를 껐다 켜달라고 친절하게 부탁한다.

    알려준대로 커맨드(⌘ Command) 키와 알파벳 q(Q)키를 눌러서 iTerm2를 완전 종료하고 다시 켜주자
    • iTerm2를 재실행하면 본격적으로 powerlevel10k 설정이 시작된다.

    또 다시 갑자기 분위기 설문조사가 시작된다.
    ---> 사이의 아이콘이 잘 보이는지 묻는다. <----
    폰트가 잘 설치 됐는지 확인하는 단계이다. 잘 보인다면 빠르게 알파벳 y(Y)를 눌러서 넘어가자.
    만약 선택을 잘못했다면 알파벳 r(R)키를 누르면 처음부터 다시 시작할 수 있다.

    자물쇠

    소용돌이 같이 생긴 데비안 로고

    git,git마스코트,시계,다이아,집,폴더,폴더(outline),속도계?
    다양한 아이콘이 있다. 이 아이콘과 X가 서로 안겹치고 잘 나오냐고 묻는다.
    • 이제 본격적으로 Prompt Style을 설정하는 단계이다. 본인이 원하는 스타일의 번호를 눌러 진행하자.

    나는 3번의 Rainbow로 설정할 것이다.

    Character Set 선택. 나는 유니코드로 선택.

    각 줄 사이를 어떤 넓이로 할 것인지

    아이콘 적게 vs 많이

    간결하게 vs TMI

    헤더와 꼬리를 입력된 커맨드에도 띄울건지 vs 안 띄울건지

    Instant Prompt 모드를 쓸 것인지 물어본다.
    이게 무엇인지 해서 읽어봤더니, zsh에서 플러그인이나 테마등을 로드하는데 시간이 오래 걸리는 경우가 있기때문에,
    로드를 다 하고 시작할 것인지, 일단 시작하고 로드할 것인지(Instant Prompt)를 물어 보는 것 같더라.
    1번을 권장한다고 해서 선택하고 넘어갔다.

    이미 powerlevel10k 설정이 있다고 뒤집어 쓸 것인지 물어보는 내용이다.
    아마 첫 설치라면 이게 안뜰텐데, 나의 경우는 블로그에 글을 쓰기 위해 처음부터 다시 깔았는데,
    환경설정 파일이 남아있어서 뜨는 것 같다. 그래서 뒤집어 쓰게 하고 넘어갔다.

    변경할 내용을 ~/.zshrc에 적용할지 물어본다.
    그렇게 많은 설정을 하게 해놓고 당연한걸 물어본다.
    여기서 no를 하면 본인이 직접 ~/.zshrc에 설정한 내용을 써주고 저장 해줘야한다.
    본인이 고통을 즐기는 편이 아니라면 알파벳 y(Y)를 눌러주자.

    powerlevel10k 적용까지해서 iTerm2 설정이 모두 완료되었다.
    조금 귀찮았지만 아주 예쁜 터미널이 완성되었다!
    역대급으로 긴 내용이지만, 사실 설정 하나하나를 다 스크린샷으로 담았기 때문에 그런것이고,
    실제로는 얼마 걸리지 않는다. 차근차근 따라해보자!
